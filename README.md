# 🌍 CapeConnect AI

**CapeConnect AI** is an AI-powered tourism assistant designed to help visitors explore Cape Town, South Africa. The application combines Artificial Intelligence with modern web technologies to provide personalized travel planning, research assistance, itinerary generation, travel email creation, and an interactive AI travel assistant.

This project was developed by **Tlangelani Chauke**, a third-year ICT Application Development student at the Cape Peninsula University of Technology (CPUT), as part of the **CAPACITI AI Skill Accelerator Programme**.

---

# 📌 Project Overview

Planning a trip to a new city can be overwhelming. Tourists often spend hours researching attractions, creating itineraries, booking accommodation, and finding reliable travel information.

CapeConnect AI simplifies this process by providing an intelligent travel companion that helps users:

- Plan personalized trips around Cape Town
- Generate professional travel-related emails
- Summarize travel itineraries and booking information
- Research attractions and travel recommendations
- Chat with an AI travel assistant
- Discover popular destinations and experiences

The application is designed with a modern, responsive interface inspired by leading travel platforms such as Airbnb and Google Travel.

# 🚀 Setup Instructions

## 1. Clone the repository

```bash
git clone https://github.com/yourusername/CapeConnect-AI.git
```

## 2. Navigate to the project

```bash
cd CapeConnect-AI
```

## 3. Install dependencies

```bash
npm install
```

## 4. Configure environment variables

Create a `.env` file in the project root.

Example:

```env
VITE_OPENAI_API_KEY=your_api_key_here
```

Replace the API key with your own.

---

## 5. Start the development server

```bash
npm run dev
```

The application will run locally at:

```
http://localhost:5173
```

---

## 6. Build for production

```bash
npm run build
```

# 🤖 Responsible AI

CapeConnect AI provides AI-generated travel recommendations to assist users with planning their trips.

Users should independently verify:

- Opening hours
- Prices
- Transport schedules
- Weather conditions
- Safety information

The application should not be relied upon as the sole source of travel information.
